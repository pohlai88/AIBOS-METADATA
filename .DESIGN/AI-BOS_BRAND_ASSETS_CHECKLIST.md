## 1. Logo & Brand Mark

### 1.1 Primary Logo

- [ ] **Full Logo (Horizontal)**

  - Symbol + Wordmark + Tagline
  - Formats: SVG, PNG (transparent), PDF
  - Sizes: Standard, 2x, 3x
  - Use: Marketing materials, headers, presentations

- [ ] **Full Logo (Vertical/Stacked)**

  - Symbol above Wordmark + Tagline
  - Formats: SVG, PNG (transparent), PDF
  - Sizes: Standard, 2x, 3x
  - Use: Mobile headers, square spaces

- [ ] **Wordmark Only**

  - "AI-BOS" typography
  - Formats: SVG, PNG (transparent)
  - Sizes: Standard, 2x, 3x
  - Use: Text-heavy contexts, email signatures

- [ ] **Icon Only (Symbol)**
  - Modular block symbol
  - Formats: SVG, PNG (transparent), ICO
  - Sizes: 16px, 32px, 64px, 128px, 256px, 512px
  - Use: Favicon, app icons, compact spaces

### 1.2 Logo Variations

- [ ] **Light Mode Logo**

  - For dark backgrounds
  - Formats: SVG, PNG (transparent)
  - Sizes: Standard, 2x, 3x

- [ ] **Dark Mode Logo**

  - For light backgrounds
  - Formats: SVG, PNG (transparent)
  - Sizes: Standard, 2x, 3x

- [ ] **Monochrome (Single Color)**

  - Black version
  - White version
  - Formats: SVG, PNG (transparent)
  - Use: Single-color printing, dark mode

- [ ] **Animated Logo**
  - Blocks assembling animation
  - Formats: GIF, MP4, Lottie JSON
  - Duration: 2-3 seconds, loop
  - Use: Website hero, presentations, demos

### 1.3 Logo Guidelines Document

- [ ] **Clear Space Rules**

  - Minimum spacing around logo
  - Examples of correct/incorrect usage

- [ ] **Size Guidelines**

  - Minimum sizes for different contexts
  - Scaling rules

- [ ] **Color Usage Rules**
  - Approved backgrounds
  - Prohibited modifications

## 16. Notes & Considerations

### Brand Consistency

- All assets must align with brand values
- No deviations from color palette
- Typography must use Inter/Mono
- Icons must follow style guide

### Accessibility

- All images need alt text
- Color contrast must meet WCAG 2.1 AA
- Videos need captions
- Documents need proper structure

### Performance

- Optimize all images (WebP, compression)
- Use SVG for icons/illustrations
- Lazy load large assets
- Provide multiple sizes (responsive)
